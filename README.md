# Fugas2

Sitio web corporativo estático para **Fugadetec**, desarrollado con **HTML, CSS y JavaScript vanilla**.

Este proyecto fue pensado para construir una web visualmente profesional, responsiva, fácil de mantener y fácil de desplegar primero en **Vercel** y luego en **cPanel** o **Hostinger**, sin depender de frameworks complejos.

---

## Descripción general

Este proyecto corresponde a una refactorización del sitio de Fugadetec. La idea principal es mantener una estética moderna, técnica y premium, pero usando una base tecnológica más simple: **HTML** para la estructura, **CSS** para estilos, animaciones y diseño responsivo, y **JavaScript** para interacción y generación dinámica de galerías.

El proyecto está preparado para que las mantenciones futuras sean simples, especialmente cuando sea necesario agregar imágenes, eliminar imágenes, cambiar enlaces de redes sociales, modificar textos del sitio, actualizar la imagen principal o cambiar los correos de contacto.

---

## Objetivo del sitio

La web está diseñada para mostrar tres líneas de negocio principales:

1. **Detección de fugas**
2. **Arquitectura**
3. **Construcción**

La portada incluye una primera pantalla potente y, más abajo, tres botones inferiores que llevan a cada línea de negocio. Cada una de esas líneas cuenta con su propio **tren de imágenes**, de modo que en el futuro sea fácil agregar o quitar fotografías sin reescribir toda la web.

Además, el sitio incluye una sección de contacto con correo general, correo comercial, correo técnico, enlace a Facebook y enlace preparado para Instagram.

---

## Tecnologías utilizadas

Este proyecto usa únicamente tecnologías del navegador:

- **HTML5**
- **CSS3**
- **JavaScript Vanilla**

Se eligió esta base porque permite máxima compatibilidad con hosting tradicional, facilidad para mover el sitio entre plataformas, mantenimiento simple, buen rendimiento, código fácil de entender e independencia de frameworks pesados.
